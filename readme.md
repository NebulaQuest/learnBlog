## 🗓️ 前端进阶 & 涨薪计划（6 个月 / 24 周）

### 🧩 第 1 个月：深入 JS & TS（扎实底层）

#### 第 1 周：JS 核心机制

- 学习内容：

  - 作用域链、闭包、原型、this、执行上下文

  - 事件循环（宏任务/微任务）

- 实践任务：

  - 手写：call、apply、bind、new、instanceof

#### 第 2 周：异步与模块化

- 学习内容：

  - Promise、async/await、事件循环深入

  - ESM 与 CommonJS、import/export 差异

- 实践任务：

  - 实现一个简易的 Promise（带 then、catch）


#### 第 3 周：TypeScript 基础

- 学习内容：

  - 类型系统、接口、泛型、类型推导、类型守卫

- 实践任务：

  - 把你的一个小 React 项目用 TS 重构


#### 第 4 周：TS 进阶 + 手写工具函数

- 学习内容：

  - 条件类型、映射类型、泛型约束

- 实践任务：

  - 用 TS 写工具库（节流、防抖、事件总线）

### ⚙️ 第 2 个月：工程化与项目规范
#### 第 5 周：Vite / Webpack 基础

- 学习内容：

  - 构建流程、Loader、Plugin、Tree-Shaking

- 实践任务：

  - 用 Vite 搭建 React + TS 项目，支持 HMR

#### 第 6 周：Lint & 规范化开发

- 学习内容：

  - ESLint、Prettier、Husky、lint-staged

- 实践任务：

  - 给公司项目配置一套代码规范自动检查流

#### 第 7 周：Git Flow + CI/CD

- 学习内容：

  - Git 分支模型、Pull Request 流程

  -   GitHub Actions（自动部署）

- 实践任务：

  - 实现 push → 自动构建 → 自动部署到 Vercel


#### 第 8 周：项目模板搭建

- 学习内容：

  - 组件化思想、项目目录规范化

- 实践任务：

  - 创建一个自己的前端脚手架模板（Vite + React + TS）

### ⚛️ 第 3~4 个月：框架原理 + 性能优化
#### 第 9~10 周：React 深入

- 学习内容：

- Virtual DOM、Diff 算法、Hooks 原理、Fiber 架构

- 实践任务：

- 实现 mini-react（支持 useState）

- 学 useMemo、useCallback 优化重渲染

#### 第 11~12 周：Vue3 深入

- 学习内容：

- 响应式原理、Proxy 实现、Composition API

- 实践任务：

- 实现一个 mini-vue 响应式系统

#### 第 13 周：前端性能优化

- 学习内容：

- 首屏优化、懒加载、资源缓存

- Chrome Performance / Lighthouse 分析

- 实践任务：

- 对现有项目进行性能分析 + 优化报告

#### 第 14 周：SSR / SEO 优化

- 学习内容：

- Next.js / Nuxt.js SSR 架构原理

- 实践任务：

- 用 Next.js 构建一个博客系统（支持 SEO）

### 🧠 第 5 个月：全栈基础 + 项目部署
#### 第 15~16 周：Node.js 与接口设计

- 学习内容：

- Express / Koa / Nest.js

- RESTful API 设计、JWT 鉴权

- 实践任务：

- 写一个用户登录 + 鉴权 API

#### 第 17 周：数据库入门

- 学习内容：

- MongoDB / MySQL 基础 CRUD、ORM（Mongoose / Prisma）

- 实践任务：

- 为前端项目接入数据库，实现数据持久化

#### 第 18 周：部署与上线

- 学习内容：

- Vercel、Render、Docker 基础

- Nginx 静态资源部署

- 实践任务：

- 把项目完整部署上线

### 🚀 第 6 个月：作品集与面试准备
#### 第 19~22 周：作品项目开发

- 选择并完成 2~3 个代表性项目：

- Vue3 + TS + Vite 的后台管理系统

- React + Node + MongoDB 的全栈博客

- 组件库（封装 Button、Modal、Table 等，发布到 npm）

#### 第 23 周：输出与优化

- 整理项目亮点与难点（性能优化、工程化、架构设计）

- 撰写技术博客（掘金/CSDN）

#### 第 24 周：面试准备 & 薪资谈判

- 高频题复习：

- JS 原理题（闭包、原型、Promise）

- 框架原理题（虚拟 DOM、响应式）

- 工程化与优化题

- 模拟面试，准备简历亮点：

- “主导过工程化改造”、“做过性能优化”、“独立搭建项目架构”

### 💼 最后：涨薪落地三步

- 准备作品集网站（展示你的项目 + 技术栈）

- 简历体现深度与成果（数据可量化）

- 刷面试题 + 模拟讲项目亮点
